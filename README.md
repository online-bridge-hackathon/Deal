# Deal
A web service that accepts request by APIs and returns 1+ bridge deals 

## Usage
```
curl https://deal.prod.globalbridge.app/api/deal
```

## Build
```
mkdir build
cd build
cmake ..
make
```

## Run api-server
```
server/api-server
```

## Test
[http://localhost:8080/api/deal](http://localhost:8080/api/deal)
